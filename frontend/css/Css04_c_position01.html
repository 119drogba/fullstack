1. static (정적)

기본적인 배치임. 기본값(디폴트)으로 별도로 지정하지 않아도 되는 속성임.

3. sticky (붙어있는)

상대적 위치의 x축, y축 좌표를 기준으로 설정한 위치에 달라 붙어 있는다.

3. sticky (붙어있는)

상대적 위치의 x축 ,y축 좌표를 기준으로 설정한 위치에 달라 붙어 있는다.

4. relative (상대적)

자신이 위치해야하는 좌측상단 좌표를 기준으로 위치를 지정한다.
위치 top,right,left,bottom으로 지정 가능하다.
이동한 만큼 다른 영역을 침번한다.

5. fixed (고정)

위치 top, right, left, bottom으로 지정 가능하며, 단위는 px, %등으로 지정 가능하다. 지정한 위치에 위에 떠 있는 것처럼 고정된다.

6. relative와 absolute 같이 사용하는 경우

현재 영역은 relative영역입니다.

7. 현재 영역은 absolute영역입니다.

absolute2 div엘리먼트를 감싸고 있는 relative2 div엘리먼트의 x축,y축을 기준으로 하여 위치한다.

8. absolute (절대적)

위치 top, right, left, bottom으로 지정 가능하며, 단위는 px, %등으로 지정 가능하다. 지정한 위치에 위에 떠 있는 것처럼 고정된다. 별도의 상대적 영역 지정이 없으면 브라우저(body영역)의 x축, y축을 기준으로 하여 위치한다.
